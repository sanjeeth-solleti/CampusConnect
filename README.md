# 🎓 CampusConnect - CIET Student Companion App

**CampusConnect** is a Flutter-based mobile application built for the students of **Chalapathi Institute of Engineering and Technology (CIET)**. It acts as a student-friendly digital portal, providing quick access to academic records, fee information, and personal details — all in one place.

---

## 📱 Features

- 🔐 **Student Login**
  - Secure login using Name and Registration ID.

- 🧑‍🎓 **Student Dashboard**
  - Displays student’s Name, Registration ID, Branch, Section, and Profile Picture.

- 💰 **Fees Section**
  - Well-structured fee details for all 8 semesters across 4 years:
    - **Total**
    - **Paid**
    - **Pending**

- 📚 **Marks Page**
  - Semester-wise academic results in a clean tabular layout.

- 📅 **(Upcoming)** Attendance Tracker
  - Subject-wise attendance tracking feature (planned for future release).

---

## 🛠️ Tech Stack

- **Frontend:** Flutter
- **Backend:** Firebase Firestore
- **Authentication:** Firebase Auth
- **Storage:** Cloud Firestore
- **Languages:** Dart
